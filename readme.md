## Problem Statement
Farmers lack an early-warning system to predict financial distress risk within upcoming 90 days.

## Solution
AgriShield Lite is an AI-powered financial risk prediction tool.
Users enter agricultural and financial parameters, and the system predicts:

- Risk Score (%)
- Risk Level (LOW / MEDIUM / HIGH)

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- AI Model: Machine Learning risk prediction model
- Version Control: Git + GitHub

## How to Run

1. Clone repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python app.py
4. Open:
   http://127.0.0.1:5000
